# 🌐 Enterprise Network Design (Cisco Packet Tracer)

## 📌 Project Overview
This project demonstrates a complete enterprise-level network design using Cisco Packet Tracer.

It includes:
- Inter-VLAN Routing
- Dynamic Routing (RIP & OSPF)
- NAT for Internet Access
- DHCP for IP Allocation
- ACL for Security
- VoIP Communication between branches
- DNS, HTTP, FTP services

---

## 🏗️ Network Features

### 🔀 Inter-VLAN Routing
Allows communication between multiple departments using router-on-a-stick.

### 🌍 NAT (Network Address Translation)
Enables internal private IPs to access the internet.

### 📡 Routing Protocols
- **RIP** → Communication between HQ and Branch
- **OSPF** → Internal routing inside HQ

### 🔐 ACL (Access Control List)
Used to restrict traffic between departments.

### ☎️ VoIP
IP Phones communicate using dial-peer configuration.
